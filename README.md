# NetPractice

## Description

NetPractice is a system administration project focused on discovering and understanding computer networking fundamentals.

The goal of this project is to solve networking problems by configuring small-scale virtual networks. Each exercise presents a non-functional network that must be fixed by correctly setting IP addresses, subnet masks, and routing information.

Through 10 different levels, this project introduces key concepts such as TCP/IP addressing and network communication, helping develop a strong foundation in how networks operate.

---

## Instructions

### Running the training interface

1. Download the project archive from the intra.
2. Extract the files into a directory of your choice.
3. Execute the command: `./run.sh`
4. Open the website in the browser

This will launch the NetPractice interface in your web browser.

---

### Using the interface

- Enter your 42 login to track your progress.
- Each level presents a broken network that must be fixed.
- Modify the editable fields (IP, mask, routes...)
- Use the following buttons:
  - Check again: validate your configuration
  - Get my config: export your solution

⚠️ Before moving to the next level, always export your configuration.

---

### Submission

- You must complete 10 levels.
- Export 1 configuration file per level using "Get my config".
- Place all 10 files at the root of your Git repository.
- File names must be correct and consistent.

---

### Evaluation

During the defense:

- You will have to solve 3 random levels
- You will have limited time of 15min
- You are not allowed to use external tools
- A basic calculator (like `bc`) is allowed
